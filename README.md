# Customer-Shopping-Behaviour-Analysis
This project focuses on analyzing customer shopping behaviour to uncover purchasing patterens, customer preferences and revenue driving factors. The analysis was performed using SQL, Python (pandas) and Power BI, with the goal of delivering clear, actionable business insights through data-driven methods and interactive dashborads.

# Problem Statement
How can a company leverage consumer shopping data to identify trends, improve customer engagement and optimize marketing and product stategies.

# Tools
1. **Python** - Data Cleaning and Feature Engineering
2. **SQL** - Data Analysis
3. **Power BI** - Interactive Dashboard and Visualizations

# Workflow
1. **Data Cleaning and Validation**
   - Handled missing values and replaced thm with their median values
   - Converted the column name into snake case to ensure consistency
   - Performed feature engineering to create two new columns (Age Group, Frequency of Purchase) from the existing tables
2. **Data Analysis**
   - Analayzed the customer spending behaviour using SQL to drive the necessary business insights
3. **Visulazation**
   - Built interactive Power BI dashbords
   - Visualized KPI's such as Revenue and Sales by Category or Age Group etc.
